# FFXIV Plugins

Central Dalamud custom repository for plugins maintained by `@gipsem`.

## Custom repository URL

When this repository and its listed plugin repositories are public, add the following URL under **Dalamud Settings -> Experimental -> Custom Plugin Repositories**:

```text
https://raw.githubusercontent.com/gipsem/FFXIV-Plugins/main/pluginmaster.json
```

Each plugin remains in its own source repository and publishes independent versioned releases. Release workflows update this index automatically.
